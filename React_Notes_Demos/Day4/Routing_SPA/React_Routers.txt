React Router 
============
React router is a standard library system built on top of react to perform routing in react application using react router library.

A routing is a process in which a user is redirected to other pages based on the inputs and actions.

React router is mainly used to create single page application (SPA).

React router is used to defined several routes in react application.

When user enters url in browser address bar and if that path matches to any route in routing file then the request will forward to that perticular route.

There are three packages to use react router.

1) react-router 
---------------
	It provides core components and functions to perform react routing.

2) react-router-native 
------------------
	It is used for mobile development.

3) react-router-dom 
--------------------
	It is used for web development.

There are two components of react router.

1) <BrowserRouter>
----------------
	It is used to handle dynamic URL's.

2) <HashRouter>
----------------
	It is used to handle static URL's.

import Home from './Home';
import About from './About';
import Service from './Service';
import Error from './Error';

function App()
{
  return(
    <>
        <BrowserRouter>
            <Routes>
                <Route exact path='/' element={<Home/>}></Route>
                <Route path='/about' element={<About/>}></Route>
                <Route path='/service' element={<Service/>}></Route>
                <Route path='*' element={<Error/>}></Route>
            </Routes>
        </BrowserRouter>
    </>
  )
}
export default App;

import Home from './Home';
import About from './About';
import Service from './Service';
import Error from './Error';

function App()
{
  return(
    <>
        <BrowserRouter>
            <center>
            <Link style={{margin:"0px 50px"}} to="/"> Home </Link>    
            <Link style={{margin:"0px 50px"}} to="/about"> About </Link>    
            <Link style={{margin:"0px 50px"}} to="/service"> Service </Link> 
            </center>
            <hr/>           

            <Routes>
                <Route exact path='/' element={<Home/>}></Route>
                <Route path='/about' element={<About/>}></Route>
                <Route path='/service' element={<Service/>}></Route>
                <Route path='*' element={<Error/>}></Route>
            </Routes>
        </BrowserRouter>
    </>
  )
}
export default App;


















Bootstrap in React 
===================
Single page application gained lot of popularity in few years and many frontend frameworks introduced single page application i.e AnglarJS, VueJS and etc.

React becomes most popular javascript library and bootstrap becomes most popular CSS framework.

To use bootstrap in react we need to download bootstrap package as given below.
ex:
	npm install bootstrap

To use bootstrap in react application we need to import inside index.js file.
