JSP : Java Server Pages

JSP: It is used to design/develop  dynamic web pages.

HTML: It is used to design/develop  static web pages.

JSP Advantages:

In  Java Server Pages , we can write any HTML code and also any Java code.

hello.jsp ----> web server ----> Hello_Servlet


Servlet :  Java code(use HTML)

JSP:  HTML (use java code) 


JSP Elements/Tags:
-----------------

1.JSP Scriplet   <%   java code  %>
2.JSP Declaration  <%!   java code %>
3.JSP Directives  <%@ page  %>
demo.jsp --> jsp compiler---> Demo_Servlet.java(Auto Generated Servlet) 
				 ---> javac---> Demo_Servlet.class ----> execute


4. JSP Expression  <%=  java expression%>
5. EL Expression  ${username}
	
Scope variables:
page  // set or get data with in a page  
request // set or get data till request scope
session  // set or get data till session is available
application  // set or get data globally
